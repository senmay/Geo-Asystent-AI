 Eksport PDF po stronie przeglądarki (frontend-only)

Najszybsze i najprostsze rozwiązanie do portfolio.
Nie potrzebujesz backendu. Możesz to zrobić np. w React + Leaflet + html2canvas + jsPDF.
Jak?

    Pobierasz dane działek (np. te, które znalazłeś w analizie)

    Generujesz z nich tabelę HTML lub zwykły tekst (np. ID + powierzchnia)

    Opcjonalnie robisz screenshot mapy (np. html2canvas)

    Składasz to do PDF (np. jsPDF)

Biblioteki:

    html2canvas – robi screenshot dowolnego div-a

    jsPDF – generuje PDF z tekstu, obrazków, canvasów

Schemat kodu:

import jsPDF from 'jspdf';
import html2canvas from 'html2canvas';

const exportToPDF = async (parcelsData) => {
  const doc = new jsPDF();

  // Dodaj tytuł
  doc.text("Raport działek", 10, 10);

  // Dodaj dane działek
  let y = 20;
  parcelsData.forEach((parcel, index) => {
    doc.text(`${index + 1}. ID: ${parcel.id} - Powierzchnia: ${parcel.area} ha`, 10, y);
    y += 10;
  });

  // Opcjonalnie screenshot mapy
  const mapElement = document.getElementById('map'); // ID kontenera mapy
  const canvas = await html2canvas(mapElement);
  const imgData = canvas.toDataURL('image/png');
  doc.addPage();
  doc.addImage(imgData, 'PNG', 10, 10, 180, 120);

  // Pobierz PDF
  doc.save('raport_dzialek.pdf');
};

Plusy:

    Zero backendu

    Działa lokalnie w przeglądarce

    Łatwe do wrzucenia do portfolio

Minusy:
